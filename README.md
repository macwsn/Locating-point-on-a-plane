# Locating a Point on a Plane Using the Trapezoidal Method

The project focuses on determining the location of a point on a map using the trapezoidal method. The program is written in Python and implemented in Jupyter Notebook.

## Repository Contents

- `code.ipynb` – The main program file performing calculations using the trapezoidal method.
- `Examples/py` – Example input data for testing the program.
- `README.md` – Main page.
- `Presentation.pttx` – Presentation.
- `Documentation.docs` – Documentation.
- `/gifs` – GIFs visualizing the creation of the trapezoidal map.
- `/requirements` – Requirements, environment setup, and recommended run instructions.
- `/requirements/visualizer.py` – Visualizer.

## Requirements

Requirements are available in the `/requirements/requriments.txt` file.

## Running the Program

Refer to the `/requirements/enviroment.txt` file.
